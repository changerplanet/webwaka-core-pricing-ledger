# Pricing Ledger

## Overview
Core module for managing pricing data and ledger operations in the WebWaka ecosystem. This is currently a substrate-only repository containing governance files and module manifest.

## Project Structure
- `index.js` - Simple HTTP server displaying module information
- `module.manifest.json` - Module metadata and configuration
- `module.contract.md` - Module contract specification
- `package.json` - Node.js package configuration

## Running the Application
The application runs on port 5000 using Node.js. Start with:
```
node index.js
```

## Development Notes
- Module ID: webwaka-core-pricing-ledger
- Class: core
